# Work Salary Bot

Личный Telegram-бот для учёта рабочего времени, перерывов, примерной зарплаты и прогресса к финансовой цели.

Бот рассчитан на обычный локальный запуск и публикацию на Railway без Docker. Локально используется SQLite, на Railway — PostgreSQL.

## Возможности

- фиксация прихода и ухода;
- несколько перерывов внутри одной смены;
- смена через полночь с корректным распределением времени по дням;
- календарь рабочих дней, выходных, отпуска, больничного и отгула;
- расчёт нормы, баланса, переработки и недоработки;
- три правила переработки: дневное, месячное и по текущему балансу;
- ставка из зарплаты, из полной выплаты или заданная вручную;
- отдельные коэффициенты будней, суббот, воскресений и праздников;
- прогноз зарплаты и текущая заработанная сумма;
- цель по доходу и реалистичный план её достижения;
- ограничения на будничную переработку и рабочие субботы;
- прогноз выплат 15-го числа и в конце месяца;
- история, ручное добавление и редактирование смен;
- мягкое удаление и восстановление последней удалённой смены;
- CSV-экспорт и JSON-копия данных;
- напоминание о приходе, уходе, открытой смене и перерыве;
- недельный отчёт;
- постоянное FSM-состояние в базе данных;
- Alembic-миграция;
- JSON-журнал и healthcheck для Railway;
- ограничение доступа по Telegram ID.

## Быстрый запуск на Windows

Установите Python 3.12 с отмеченным пунктом `Add python.exe to PATH`.

В корне проекта выполните:

```powershell
.\setup.ps1
```

При запрете выполнения PowerShell-скриптов один раз выполните:

```powershell
Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
```

Альтернативный вариант:

```powershell
setup.bat
```

После установки откройте `.env` и заполните:

```env
BOT_TOKEN=токен_от_BotFather
ALLOWED_USER_IDS=ваш_telegram_id
ADMIN_USER_ID=ваш_telegram_id
DEFAULT_TIMEZONE=Europe/Istanbul
DATABASE_URL=sqlite+aiosqlite:///data/bot.db
```

Запуск:

```powershell
run.bat
```

Или напрямую:

```powershell
.\.venv\Scripts\Activate.ps1
python main.py
```

Миграция выполняется автоматически при старте. При необходимости её можно запустить вручную:

```powershell
alembic upgrade head
```

## Получение Telegram ID

Временно оставьте `ALLOWED_USER_IDS` пустым при локальном запуске, откройте бота и отправьте:

```text
/myid
```

После получения ID заполните `ALLOWED_USER_IDS` и перезапустите бота.

На Railway пустой список разрешённых пользователей блокирует запуск приложения.

## Основное меню

```text
📍 Сегодня
🟢 Пришёл      🔴 Ушёл
☕ Перерыв     📊 Анализ
🗓 История     📅 Календарь
💳 Выплаты     📤 Экспорт
⚙️ Настройки
```

Полный анализ не отправляется после каждого действия. Кнопки прихода, ухода и перерыва возвращают короткое подтверждение.

## Расчёт зарплаты

Бот разделяет три показателя:

- заработано сейчас — пропорциональная часть фиксированной выплаты и уже рассчитанная переработка;
- прогноз месяца — полная фиксированная выплата с учётом текущей недоработки и переработки;
- план цели — дополнительное время, необходимое до выбранного дохода.

Расчёт является ориентировочным. Итоговая сумма зависит от правил работодателя, поэтому профиль оплаты нужно настроить под реальные условия.

## Стиль комментариев

Комментарии в Python-коде оформлены короткими именными конструкциями:

```python
# расчёт месяца
# профиль пользователя
# подключение базы
```

Используется именительный падеж и единственное число. Комментарии с описанием действия в повелительной или глагольной форме не применяются.
